# システム基本設計書

**Project:** Ikotsu.com Next-Generation Platform
**Version:** 1.0
**Author:** Seiren Group

---

## 1. システム概要

粉骨・洗骨を中心とした遺骨整備専門サイト。BtoC・BtoB双方を対象とする。

**主目的：** SEO上位獲得 / GEO最適化 / 高CV率 / 法人契約獲得 / 清蓮グループ連携強化

**技術前提：** Next.js 16 / App Router / SSR対応 / SEO最適化

---

## 2. 全体アーキテクチャ

| 区分           | 方針                                 |
| -------------- | ------------------------------------ |
| フロントエンド | Next.js App Router                   |
| 主要ページ     | SSR                                  |
| ブログ記事     | SSG                                  |
| 状態管理       | 料金シミュレーターはClient Component |
| フォーム送信   | API Route経由                        |

---
