# 被リンク戦略・デジタルPR設計書

**Project:** Ikotsu Lab Backlink Strategy & Digital PR Design
**Layer:** SEO Authority Growth
**Version:** 1.0
**Purpose:** 粉骨・洗骨分野で検索上位を安定化させるための被リンク戦略および権威構築設計

---

## 1. 設計目的

内部SEOだけでは限界がある。上位安定には権威 / 外部評価 / 引用が必要。
**目的：** ドメイン評価向上 / AI引用増加 / 法人信頼強化（量より質）

---

## 2. 被リンク方針

**NG：** リンク購入 / 低品質ディレクトリ登録 / スパム投稿
**OK：** 専門性ベースの自然獲得 / 提携先紹介 / 共同コンテンツ制作

---

## 3. 獲得ターゲットカテゴリ

寺院サイト / 納骨堂サイト / 終活メディア / 法律解説メディア / 葬儀関連メディア / 大学・研究機関記事（将来）
YMYLに近い分野は信頼重視。

---

## 4. 具体施策

### 4.1 寄稿戦略

テーマ例：遺骨のカビ問題 / 粉骨の法的整理 / 納骨堂湿気対策（専門記事を提供）

### 4.2 共同記事制作

寺院 × Ikotsu Lab / 散骨業者 × Ikotsu Lab（双方掲載）

### 4.3 事例公開

法人提携先紹介ページ / 提携事例掲載（自然リンク誘発）
