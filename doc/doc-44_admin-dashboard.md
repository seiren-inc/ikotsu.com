# 管理画面・バックオフィス設計書

**Project:** Ikotsu.com Admin Dashboard & Backoffice Management Design
**Layer:** Technical / Operational Architecture
**Version:** 1.0
**Purpose:** 問い合わせ対応・法人管理・単価分析を一元化する管理画面設計

---

## 1. 設計目的

サイトが伸びても、管理が崩れたら終わる。
**目的：** 問い合わせの見落としゼロ / 対応スピード向上 / 法人優先管理 / 単価・CV分析可視化
**属人化させない。**

---

## 2. 管理画面全体構造

`/admin` 配下：ダッシュボード / BtoC管理 / BtoB管理 / 診断ログ / 見積ログ / 法人管理 / スタッフ管理 / 分析画面 / 設定

---

## 3. ダッシュボード画面

**表示項目：** 本日の新規問い合わせ数 / 未対応件数 / 高ボリューム法人件数 / 平均見積単価 / CV率
**グラフ：** 月次推移 / 法人比率 / 単価推移
トップで状況把握。

---

## 4. BtoC管理画面

**一覧表示：** 名前 / 問い合わせ種別 / 見積金額 / ステータス / 担当者 / 日時
**フィルタ：** 未対応 / 見積済 / 成約 / クローズ
ワンクリックで詳細表示。

---

## 5. BtoB管理画面

**表示項目：** 法人名 / 月間件数 / 優先フラグ / 契約状況 / 担当者
高ボリューム法人は自動色分け。

---

## 6. 診断・シミュレーター分析画面

**表示：** 診断完了率 / シミュレーター利用率 / オプション選択率 / 平均見積金額
データドリブン改善。

---

## 7. 詳細画面設計

**問い合わせ詳細：** 基本情報 / 診断履歴 / シミュレーター結果 / 内部メモ / 対応履歴 / ステータス変更
履歴は時系列表示。

---

## 8. ステータス設計

`new` → `contacted` → `quoted` → `converted` → `closed`
変更時にログ保存。

---

## 9. 権限管理設計

- **Admin：** 全権限
- **Staff：** 編集可能
- **Viewer：** 閲覧のみ
  操作ログ保存。

---

## 10. 検索・フィルタ機能

名前検索 / 法人名検索 / 価格帯検索 / 期間指定 / ステータス絞り込み（管理効率向上）

---

## 11. セキュリティ設計

二段階認証 / IP制限（将来） / 操作ログ保存 / CSRF対策
**管理画面は最重要領域。**

---

## 12. 将来拡張

法人ポータル連携 / 請求書自動生成 / PDF見積書生成 / 支払い状況管理 / 顧客マイページ連携
拡張前提設計。

---

## 13. KPI

未対応ゼロ / 平均返信24時間以内 / 法人契約率向上 / 単価上昇

---

## 14. 最終目標

問い合わせを取りこぼさない。**Webと現場を完全統合する管理基盤。**
