# BtoB戦略設計書

**Project:** Ikotsu Lab BtoB Strategy Design
**Layer:** Strategic Design
**Version:** 1.0
**Purpose:** 寺院・納骨堂・霊園・関連事業者から継続受注を獲得し、安定収益基盤を構築するための戦略設計

---

## 1. BtoB市場定義

**対象法人：**

- 寺院
- 納骨堂運営会社
- 樹木葬事業者
- 霊園管理会社
- 石材店
- 葬儀社

**市場背景：**
改葬件数増加 / 納骨堂・樹木葬の増加 / 寺院の人手不足 / 衛生管理基準の高度化 / 外注化ニーズの拡大
BtoBは単発ではなく「継続処理市場」。

---

## 2. 法人顧客の課題整理

**寺院・納骨堂側の課題：**
粉骨設備を持たない / 人員不足 / 衛生管理に不安 / 処理品質のばらつき / 保管スペース問題 / 作業リスク / 法的トラブル回避
Ikotsu Labはこれを"外部専門機関"として解決する。

---

## 3. ポジショニング戦略

**ポジション：「遺骨整備専門の外部インフラ」**

価格勝負をしない。品質と安定供給で選ばれる。

**差別化軸：**
工程標準化 / トレーサビリティ / 大量処理能力 / 緊急対応 / 法人専用窓口 / 証明書発行

---

## 4. 法人ターゲット優先順位

1. **納骨堂運営者**（改葬・粉骨需要が集中。継続案件になりやすい）
2. **樹木葬事業者**
3. **寺院**
4. **石材店**
5. **葬儀社**

---

## 5. 提携モデル設計

- **モデルA：完全外注型**（寺院が粉骨を全て委託、ロット単位で処理）
- **モデルB：共同ブランド型**（「◯◯寺 提携粉骨サービス」として展開。収益分配モデル）
- **モデルC：年間契約型**（月間最低件数契約。単価優遇）
- **モデルD：緊急スポット型**（突発案件のみ対応）

---

## 6. 価格戦略

**価格は公開しない。個別見積。**

**戦略：**

- 月間20体以上 → 単価優遇
- 月間50体以上 → さらに優遇
- 年間契約 → 最優遇

価格競争を回避し、「安定性と信頼性」で契約する。

---

## 7. 営業チャネル戦略

- **直接営業：** 電話アポ / DM送付 / 紹介営業
- **間接営業：** 業界団体 / 石材店ネットワーク / 清蓮グループ経由
- **コンテンツ営業：** 法人向けSEO記事 / 外注のメリット解説 / 無料資料DL

---

## 8. 法人専用サイト設計戦略

**法人専用ページに必須：**

- 課題提示 / 処理工程 / 品質管理体制 / 証明書サンプル / 導入フロー / テスト導入提案 / 法人専用フォーム / 資料DL
  資料DLでリード取得。

---

## 9. 収益シミュレーション

**例：**

- 1社あたり月間15体 × 平均単価15,000円 → 月225,000円
- 10社契約 → **月2,250,000円**
  安定収益基盤。

---

## 10. KPI設計

- **月間新規接触：** 50件
- **アポ率：** 10%
- **契約率：** 20%
  月1〜2社契約獲得。

---

## 11. 中期目標

**6ヶ月以内：** 法人5社契約
**12ヶ月以内：** 法人15社契約
**最終目標：** 遺骨整備の外注標準インフラとなる。
