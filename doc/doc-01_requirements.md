# 要件定義書

**Project:** Ikotsu Lab Next-Gen Platform
**Version:** 1.0
**Goal:** 遺骨サービス業界で最も先進的で信頼性の高いWeb基盤構築

---

## 1. ヘッダー機能要件

### 必須ナビ項目

- 粉骨サービス
- 洗浄サービス
- 運搬サービス
- 骨壺取り出し
- 料金シミュレーター
- ご利用の流れ
- よくある質問
- お客様の声
- 会社情報
- 無料相談

### 常時固定要素

- 電話番号（クリック即発信）
- LINE相談ボタン
- 無料見積ボタン
- 緊急対応表示（例：最短◯日）

### スマホ専用固定バー

- 電話
- LINE
- 見積

### メガメニュー機能

ホバー時に表示：

| ナビ項目 | 展開内容   |
| -------- | ---------- |
| 粉骨     | 用途別     |
| 洗浄     | 状態別     |
| 運搬     | 距離別     |
| 取り出し | 現場対応例 |

## 2. 自動機能（コア機能）

### ① 料金自動見積エンジン

- リアルタイム計算
- 内訳自動表示
- フォーム連動
- 条件保存（Cookie）

### ② AI最適プラン提案機能

ユーザー選択に基づき「あなたに最適なのは◯◯プランです」と提案。

**ロジック例：**
土付着 + 改葬予定 → 洗浄＋粉骨推奨

### ③ 動的FAQ生成

選択内容に応じて関連FAQを自動表示。

### ④ ステップ進行UI

- 現在位置表示
- 残りステップ可視化
- 心理負担軽減

### ⑤ 自動安心メッセージ表示

**例：** 墓納骨選択時 → 「湿気除去処理を推奨します」

## 3. 先進的モダン機能

### ① デジタル作業レポート機能（将来実装）

- 作業前後写真共有
- 処理工程レポートPDF自動生成
- マイページログイン

### ② リアルタイム対応可否表示

管理画面と連動し「本日受付可能」「今週満枠」などを表示。

### ③ 安心スコア表示

透明性指標：

- 作業実績数
- 平均対応日数
- 顧客満足度

### ④ 進行状況トラッキング

申込後の状況（受付 → 作業中 → 完了 → 発送）を可視化。

### ⑤ 即時概算PDF発行

見積結果をワンクリックでPDF生成。

### ⑥ チャット相談（AI + オペレーター切替）

- 営業時間外 → AI対応
- 営業時間内 → 人間対応

### ⑦ 遺骨状態診断フロー

質問形式で「あなたの遺骨はどの処理が必要か」を診断表示。

### ⑧ サービス組み合わせ最適化UI

複数サービス選択時に割引提案を表示。
