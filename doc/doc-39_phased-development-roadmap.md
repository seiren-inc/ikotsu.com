# 実装ロードマップ・段階的開発計画

**Project:** Ikotsu Lab Implementation Roadmap & Phased Development Plan
**Layer:** Execution Planning
**Version:** 1.0
**Purpose:** 戦略・設計書群を実装可能な工程へ落とし込み、Antigravityで確実に構築するための実装計画

---

## 1. 実装前提

目的は「美しいサイト」ではない。
・SEOトップ獲得 / CV最大化 / 法人獲得 / ブランド確立
**戦略 → 設計 → 実装 → 改善** の流れで進める。

---

## 2. フェーズ設計

- **Phase 1：** 基盤構築
- **Phase 2：** CV装置実装
- **Phase 3：** 権威性強化
- **Phase 4：** 法人拡張
- **Phase 5：** 高度最適化

段階的に構築。

---

## 3. Phase 1：基盤構築

**実装項目：** トップページ / 粉骨ページ / 洗骨ページ / 法人ページ / ガイドページ / 会社情報 / 法的ページ / 内部リンク構造 / 基本SEO
**目標：** 最低限の公開可能状態。
**期間目安：** 2〜3週間

---

## 4. Phase 2：CV装置実装

**実装項目：** AI診断 / 料金シミュレーター / BtoCフォーム / BtoBフォーム / 固定CTA / イベント計測
**目標：** CV率改善。
**期間目安：** 2週間

---

## 5. Phase 3：権威性強化

**実装項目：** 専門解説記事群 / FAQ強化 / 構造化データ / E-E-A-T設計 / 著者情報表示
**目標：** 検索順位向上。
**期間目安：** 継続実装

---

## 6. Phase 4：法人拡張

**実装項目：** 法人専用資料DL / 商談予約機能 / 法人導線最適化 / 法人記事拡充
**目標：** 法人比率向上。
**期間目安：** 1ヶ月

---

## 7. Phase 5：高度最適化

**実装項目：** LTV分析基盤 / PDF自動生成 / マイページ構想 / CRM統合 / 単価最適化ロジック
**目標：** 供養インフラ化。

---

## 8. 優先順位

- **最優先：** 粉骨・洗骨ページ / 診断 / シミュレーター
- **次点：** 法人ページ / SEO記事
- **後回し：** マイページ / 高度自動化

---

## 9. KPIマイルストーン

- **公開1ヶ月：** 検索流入獲得開始
- **3ヶ月：** 主要KW 20位以内
- **6ヶ月：** 10位以内 / 法人問い合わせ月3件以上

---

## 10. Antigravity実装方針

- 設計書単位で実装指示
- コンポーネント分割
- 最小差分実装
- 段階的デプロイ
- Lighthouse確認
  **大きく一気に作らない。**

---

## 11. リスク管理

**想定：** 工数超過 / 機能肥大化 / SEO反映遅延
**対策：** Phase分割 / 優先順位固定 / 毎月レビュー

---

## 12. 最終目標

Ikotsu Labを、粉骨・洗骨分野の標準サイトにする。
戦略から実装まで一貫管理。
