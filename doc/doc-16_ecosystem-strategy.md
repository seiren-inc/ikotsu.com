# エコシステム統合戦略設計書

**Project:** Ikotsu Lab Ecosystem Integration Strategy Design
**Layer:** Strategic Design
**Version:** 1.0
**Purpose:** 清蓮グループ全体のサービスと遺骨ラボを統合し、単発受注から循環型収益モデルへ進化させる戦略設計

---

## 1. エコシステム戦略の定義

Ikotsu Labは単体事業ではない。

**役割：** 清蓮グループの入口インフラ。
遺骨整備を起点に、次の供養サービスへ自然に接続する。

**エコシステム対象：**

- 散骨クルーズ
- お墓探しナビ
- お墓じまいナビ
- 遺骨ダイヤモンドアドバイザー
- 終活コンシェルジュ
- 清蓮コーポレート

**目的：** LTV最大化とブランド一貫性の確立。

---

## 2. 顧客ジャーニー統合設計

**BtoC統合フロー：**
遺骨状態不安 → 粉骨・洗骨 → 次の供養選択 → 散骨 / 墓じまい / 自宅供養 / ダイヤモンド → 終活コンシェルジュへ拡張

**BtoB統合フロー：**
寺院・納骨堂 → 粉骨外注 → 墓じまいナビ提携 → 散骨クルーズ連携 → 継続契約

---

## 3. クロスセル導線設計

粉骨完了後ページに表示：

- 散骨という選択肢
- 納骨堂のご紹介
- 墓じまいサポート
- 遺骨ダイヤモンド加工
- 終活相談

表示は押し付けない。選択肢として提示。

---

## 4. データ統合戦略

**将来構想：**
顧客ID統合 / 問い合わせ履歴共有 / クロスサービス履歴連携 / 紹介トラッキング

Ikotsu Labをデータハブにする。

---

## 5. ブランド統合戦略

**トーン統一：** 清潔感 / 専門性 / 安心感
グループロゴ統一 / フッター統一 / 相互リンク設計

---

## 6. 収益循環モデル

**例：**
粉骨 25,000円
→ 散骨 150,000円
→ 墓じまい 300,000円
→ 終活相談 50,000円

入口は小さく、出口は大きく。**LTV 10倍設計。**

## 8. 法人エコシステム

**寺院と提携：**
粉骨外注 → 墓じまい案件紹介 → 散骨案件紹介 → 相互利益モデル
Win-Win構造。

- **クロスセル率：** 30%以上

- **グループ内送客：** 月20件以上
