# マスター要件定義書（統合版）

**Project:** Ikotsu Lab Next-Generation Platform
**Version:** 1.0
**Goal:** 粉骨・洗骨業界でSEO/GEO/BtoB/BtoCすべてを制するトップサイト構築

> [!IMPORTANT]
> 本書は全設計書の統合版マスタードキュメント。他の個別設計書と合わせて参照すること。

---

## 1. プロジェクト目的

遺骨ラボを以下のポジションへ進化させる：

- 遺骨整備の専門機関
- 清蓮グループのハブ
- SEO上位サイト
- 生成AI参照サイト
- 法人外注インフラ

単なるサービスサイトではなく**業界標準プラットフォーム**を目指す。

---

## 2. ターゲット定義

| 市場 | ターゲット                                                                           |
| ---- | ------------------------------------------------------------------------------------ |
| BtoC | 改葬予定者 / 墓じまい検討者 / 散骨希望者 / 自宅供養希望者 / 遺骨の状態に不安がある人 |
| BtoB | 寺院 / 納骨堂 / 樹木葬運営者 / 霊園 / 石材店 / 葬儀社                                |

## 4. 必須機能要件

### 料金自動見積エンジン

- 状態別・サイズ別・距離加算・オプション加算で計算
- 内訳表示 / PDF出力 / フォーム連携

### 遺骨状態診断機能

- 質問形式UI → 最適プラン提案

### AIレコメンド

- 選択内容に応じて関連サービスを動的表示

### BtoB専用フォーム

- ロット入力 / 法人資料DL / 担当直通導線

### クロスサービス導線

| サービス名         | リンク先           |
| ------------------ | ------------------ |
| 散骨               | 散骨クルーズ       |
| お墓探しナビ       | グループサービス   |
| お墓じまいナビ     | グループサービス   |
| 遺骨ダイヤモンド   | グループサービス   |
| 終活コンシェルジュ | グループサービス   |
| 清蓮コーポレート   | コーポレートサイト |

## 5. SEO要件

- ピラーページ：「遺骨整備完全ガイド」構築
- ロングテール記事：月4本以上
- 構造化データ：FAQ schema / LocalBusiness schema / Breadcrumb schema
- Core Web Vitals最適化 / 画像WebP / LCP 2秒以内

**目標：** 「粉骨 料金」で上位3位以内

---

## 6. GEO要件（生成AI引用対策）

各ページに必須：

- 定義文（「粉骨とは〜」形式）
- 比較表
- Q&A明確化
- 数値提示
- 実績表示
