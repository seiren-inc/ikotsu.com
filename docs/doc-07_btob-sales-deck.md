## スライド構成

### 1. 表紙

- タイトル：粉骨・洗骨 専門パートナーのご提案
- サブ：寺院・納骨堂・霊園様向け 外部委託モデル
- ロゴ：Ikotsu.com × 清蓮グループ

---

### 2. 問題提起

**寺院・霊園様が抱える課題：**

- 人手不足
- 衛生管理負担
- 設備コスト
- 保管スペース問題
- 改葬増加への対応
- 法的リスク

データ提示：改葬件数増加トレンド

---

### 3. 解決策提示

Ikotsu.comは「遺骨整備専門機関」として外部パートナー機能を提供。

**提供内容：** 粉骨 / 洗骨 / 土泥除去 / 出張回収 / 大量ロット対応 / 緊急対応

---

### 4. 処理工程

```
受領 → 異物除去 → 乾燥処理 → 粉骨加工 → 封入 → 検品 → 返送
```

写真付きで清潔感を強調。

---

### 5. 品質管理体制

- 衛生基準明示
- 作業室写真
- 処理証明書発行
- トレーサビリティ管理
- 保険加入

---

### 6. 法人向けメリット

- 設備投資不要
- 人件費削減
- 作業リスク軽減
- 作業時間短縮
- 顧客満足度向上

---

### 7. 価格モデル（概要）

法人価格は個別見積制（公開しない）。

- ロット割引あり
- 年間契約優遇
- 月間処理数に応じて単価調整

---

### 8. 導入フロー

```
① ヒアリング → ② 試験導入（少量） → ③ 正式契約 → ④ 定期回収
```
