# メールナーチャリング・フォローアップ自動化設計書

**Project:** Ikotsu.com Email Nurturing & Follow-up Automation Design
**Layer:** Growth / LTV Design
**Version:** 1.0
**Purpose:** 問い合わせ後の放置を防ぎ、成約率と単価を引き上げるフォローアップ設計

---

## 1. 設計目的

遺骨サービスは即決しない（家族と相談 / 日程調整 / 費用検討）。
**目的：** 成約率向上 / 単価向上 / 法人継続化 / 信頼維持

---

## 2. BtoCフォロー設計

| タイミング | 内容                                  |
| ---------- | ------------------------------------- |
| 即時       | 受付確認 / 安心メッセージ             |
| 24時間後   | 工程解説 / よくある不安               |
| 3日後      | 料金内訳再提示 / 洗骨提案（条件付き） |
| 7日後      | 最終確認 / 電話相談案内               |

押し売りしない。

---

## 3. BtoBナーチャリング設計

| タイミング | 内容                            |
| ---------- | ------------------------------- |
| 即時       | 受付確認                        |
| 3日後      | 法人事例紹介                    |
| 1週間後    | コストモデル提示 / 湿気問題解説 |
| 1ヶ月後    | 提携メリット再提示              |

継続接触。

---

## 4. メール構成原則

短い / 専門的 / 誠実 / 売り込み過ぎない。文章は簡潔。

---

## 5. セグメント設計

**セグメント：** 散骨予定者 / 改葬予定者 / カビ不安者 / 法人高ボリューム
内容を変える。

---

## 6. 自動化設計

**トリガー：** フォーム送信 / 診断完了 / シミュレーター利用 / 資料DL
CRM連携前提設計。

---

## 7. 単価向上設計

メール内で：保存対策説明 / 粒度指定説明 / 証明書価値説明
**教育型アップセル。**

---

## 8. KPI

成約率向上 / 平均単価向上 / 法人再接触率向上 / 開封率30%以上

---

## 9. 禁止事項

連投 / 過度な割引 / 煽り文 / 緊急性演出
**信頼を壊さない。**

---

## 10. 将来拡張

SMS連携 / LINE連携 / 法人専用ニュースレター / 定期レポート送付

---

## 11. 最終目標

問い合わせで終わらせない。**信頼を積み上げるフォロー設計。**
