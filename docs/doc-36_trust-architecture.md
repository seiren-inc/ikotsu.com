# 信頼性・E-E-A-T強化設計書

**Project:** Ikotsu.com Trust Architecture & E-E-A-T Reinforcement Design
**Layer:** Strategy Design
**Version:** 1.0
**Purpose:** 粉骨・洗骨分野においてGoogleおよび生成AI双方から「最も信頼できる情報源」と評価されるための信頼設計

---

## 1. 設計目的

遺骨というテーマはYMYL領域に近い。
不安・法律・宗教・倫理が絡むため、**E-E-A-T（経験、専門性、権威性、信頼性）を意図的に設計する。**

---

## 2. Experience（経験）設計

**実体験・実務経験の可視化：**
実際の工程写真 / ビフォーアフター例 / 法人対応事例 / 具体的な処理フロー解説 / 現場写真（衛生管理含む）
抽象論ではなく実務公開。

---

## 3. Expertise（専門性）設計

**専門性の証明：**
粉骨粒度の科学的説明 / 湿気・カビの原因解説 / 保存理論 / 改葬と粉骨の法的整理 / 数値データ提示
文章は専門的だが難解にしない。

---

## 4. Authoritativeness（権威性）設計

**権威性の構築：**
法人提携掲載 / 実績数表示（誇張しない） / メディア掲載実績（将来） / 関連サービスとの連携明示 / 会社情報の透明化
清蓮コーポレートとの明確な接続。

---

## 5. Trustworthiness（信頼性）設計

**最重要要素：**
料金明示 / 追加請求なし明記 / キャンセル規定明示 / プライバシーポリシー明確化 / 特商法表記完全整備 / 問い合わせ先明示
不透明な業者との差別化。

---

## 6. 著者情報設計

**全専門記事に：** 監修者名 / 肩書き / 所属 / 簡単な経歴 / 関連資格（あれば）
匿名記事は作らない。

---

## 7. FAQ構造強化

全サービスページにFAQ設置。（例：粉骨は違法ですか？ / 自宅で砕くのは可能ですか？ / 湿気はどう防ぐ？）
**FAQPage構造化データ実装。**

---

## 8. 外部評価対策

将来施策：専門コラム寄稿 / 寺院との共同記事 / 学術的引用リンク / 信頼性の高い外部リンク
被リンクは量より質。

---

## 9. AI検索対策

LLMが引用しやすい構造：定義文を明確に / 簡潔なまとめ段落 / 工程説明を箇条書き化 / 数字を明示 / 誤解を招く断定を避ける
**ChatGPTやPerplexityで引用される設計。**

---

## 10. 信頼阻害要因の排除

**排除：**過剰演出 / 強すぎる煽り / 価格だけの訴求 / 他社批判 / 曖昧な表現
**静かな専門性。**

---

## 11. KPI

指名検索増加 / AI引用増加 / 検索順位安定 / 問い合わせ時の価格交渉減少

---

## 12. 最終目標

**粉骨・洗骨の基準サイトになる。**
専門性と透明性で市場を制圧する。
