# サービスナビゲーション構造設計書

**Project:** Ikotsu.com Service Navigation Structure
**Goal:** 情報整理とCV最大化を両立するサービス構造設計

---

## 1. 設計思想

トップページは説明過多にしない。サービスはプルダウン式で分類。詳細は専用ページで深掘る。

**「迷わせない」「すぐ選ばせる」構造。**

---

## 2. ヘッダー構造

| 項目                       |
| -------------------------- |
| サービス一覧（プルダウン） |
| 料金シミュレーター         |
| 個人のお客様               |
| 法人のお客様               |
| 遺骨整備完全ガイド         |
| 清蓮グループ               |
| 無料相談                   |

---

## 3. サービスプルダウン内容

| サービス名           | URL                   | 説明文                     |
| -------------------- | --------------------- | -------------------------- |
| 粉骨サービス         | `/services/powdering` | 遺骨をパウダー状に整えます |
| 洗骨・洗浄サービス   | `/services/cleaning`  | カビ・汚れを専門洗浄します |
| 遺骨お運びサービス   | `/services/transport` | 全国対応の出張・搬送       |
| 骨壺取り出しサービス | `/services/takeout`   | 骨壺からの取り出し対応     |
| 出張粉骨サービス     | `/services/on-site`   | 現地での粉骨作業           |

> [!NOTE]
> 現在の実装では `/services/carrying`（運搬）が `/services/transport` に相当する。URL変更または追加が必要。
> 出張粉骨 `/services/on-site` は未実装。

## 4. 各サービスページ共通テンプレート構造

1. Heroセクション
2. サービス概要
3. 対象となるケース
4. 処理工程説明
5. 料金表
6. 料金シミュレーターへのリンク
7. よくある質問
8. 次に選べる供養方法
9. CTA固定

---

## 5. URL設計

```
/services/powdering   # 粉骨
/services/cleaning    # 洗骨
/services/transport   # 運搬・搬送
/services/takeout     # 骨壺取り出し
/services/on-site     # 出張粉骨
```

SEO最適化のため英語スラッグ使用。

---

## 6. UI設計

- プルダウンはメガメニュー型
- スマホではアコーディオン式
- 現在ページは強調表示
- CTAボタンは常時右下固定

---

## 7. UX意図

```
サービス選択 → 専用ページ → シミュレーター → 相談
```

迷いを排除する。

---

## 8. SEO意図

- トップはハブ
- 各サービスページが主戦場（単独で上位表示させる）

---

## 9. 将来拡張

サービス追加時はプルダウンに追加するだけ。拡張性確保。
