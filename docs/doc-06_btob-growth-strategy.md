## 3. 提案モデル設計

### モデル1：外注型

寺院が粉骨設備を持たない場合 → 完全外注

提案内容：定期回収・まとめ処理・返送・証明書発行

### モデル2：共同ブランド型

「◯◯寺 提携粉骨サービス」として提供。寺院は手間ゼロ。収益分配モデル。

### モデル3：大量ロット契約型

- 月間20体以上処理 → 単価優遇
- 年間契約 → さらに割引

---

## 4. 営業アプローチ設計

```
フェーズ1 接触
  方法：DM送付 / 電話アポ / 紹介 / 業界団体経由 / セミナー開催
    ↓
フェーズ2 ヒアリング
  確認：月間処理件数 / 現在の処理方法 / 外注有無 / 課題
    ↓
フェーズ3 提案
  資料：処理工程 / 衛生管理基準 / 価格モデル / 実績 / 保証
    ↓
フェーズ4 テスト導入
  小ロット（5体程度）から導入
    ↓
フェーズ5 年間契約
  安定供給体制へ
```

## 5. 価格戦略（法人）

個別見積制（公開しない）。

| 条件         | 優遇       |
| ------------ | ---------- |
| 月間20体以上 | 10%優遇    |
| 月間50体以上 | 15%優遇    |
| 年間契約     | さらに優遇 |
