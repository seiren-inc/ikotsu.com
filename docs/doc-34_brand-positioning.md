# ブランドポジショニング・市場差別化設計書

**Project:** Ikotsu.com Brand Positioning & Market Differentiation Design
**Layer:** Strategy Design
**Version:** 1.0
**Purpose:** 価格競争から脱却し、粉骨・洗骨分野で「専門機関ポジション」を確立するためのブランド戦略設計

---

## 1. 市場前提整理

遺骨サービス市場の特徴：価格差が大きい / 品質の可視化が難しい / 法律理解が不十分な業者が存在 / 不安商材である
多くの業者は「安い」「即日」「簡単」を前面に出している。
**Ikotsu.comはそこに行かない。**

---

## 2. ブランドポジション定義

**ポジション：** 粉骨・洗骨の専門機関
**キーワード：** 安全性 / 透明性 / 科学的根拠 / 工程公開 / 法人対応
安さではなく、安心で選ばれる。

---

## 3. 差別化軸設計

Axis 1：工程公開レベル
Axis 2：料金透明性
Axis 3：法人実績
Axis 4：衛生管理
Axis 5：相談体制
**これらをサイト上で明示。**

---

## 4. ブランドメッセージ設計

**コアメッセージ：** 遺骨処理を、専門技術として扱う。
**サブメッセージ：** 工程を公開しています / 追加請求はありません / 科学的処理を行います / 法人委託にも対応
宗教色を出しすぎない。

---

## 5. 価格ポジション戦略

最安値を目指さない。**価格帯：中〜やや上**
理由：信頼を価格で担保 / 法人案件対応 / 長期ブランド構築
**価格は「安心コスト」。**

---

## 6. BtoCポジション

**ターゲット：** 散骨予定者 / 改葬予定者 / 長期保管者 / カビ問題発生者
**価値提供：** 不安解消 / 専門説明 / 工程の可視化

---

## 7. BtoBポジション

**ターゲット：** 寺院 / 納骨堂 / 霊園 / 樹木葬事業者
**価値提供：** 外注安定化 / 品質均一化 / クレーム削減 / 湿気対策
**粉骨外注の標準窓口になる。**

---

## 8. 競合との差別化表現

**避ける：** 他社批判 / 安さ強調 / 煽り表現
**使う：** 工程比較 / 安全基準 / 法的解説 / 数値根拠
静かに強い。

---

## 9. ビジュアルブランディング

白基調 / 清潔感 / ラボ的印象 / 青系アクセント / 金は最小限
高級すぎない。医療寄りすぎない。

---

## 10. 権威性構築

法人提携実績掲載 / 工程写真公開 / FAQ充実 / 法的解説記事 / メディア掲載（将来）
**E-E-A-T強化。**

---

## 11. ブランドKPI

指名検索増加 / 法人問い合わせ増加 / 価格交渉減少 / 紹介率向上

---

## 12. 最終目標

**「安い業者」ではなく、粉骨・洗骨の基準を作る存在へ。**
